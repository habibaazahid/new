# My Portfolio Website

A personal portfolio website built from scratch as a beginner project.

Live: [myportfolio-sk7e.vercel.app](https://myportfolio-sk7e.vercel.app)

---

## What I Used

**HTML5** — page structure, semantic tags, sections, navigation  
**CSS3** — custom variables, flexbox, grid, animations, keyframes, media queries  
**JavaScript** — DOM manipulation, IntersectionObserver, event listeners, timers  
**Google Fonts** — Syne, DM Sans, DM Mono  
**Vercel** — free deployment and hosting  
**GitHub** — version control and source code  

---

## What I Built

- Custom cursor that follows the mouse with an animated trailing ring
- Scroll reveal — sections fade in as you scroll down the page
- Skill bars that animate and fill up when scrolled into view
- SVG circular progress rings with gradient strokes
- Typing animation that cycles through different text
- Project cards with a radial glow that follows the mouse
- Stat cards with a 3D tilt effect on hover
- Certification cards that fade in one by one
- Navigation bar that shrinks and darkens when scrolling
- Fully responsive layout for mobile and desktop
